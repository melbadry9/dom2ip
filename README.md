# dom2ip

Resolve domain to ip and print the result to output stream.

## Installation

- `git clone https://github.com/melbadry9/dom2ip.git`
- `pip3 -r install requirements.txt`

## Usage

- `python3 d2ip.py file.txt`
- `python3 d2ip.py domain.com,example.com`
